# Tailwind CSS Assignment

## Objective

The goal of this assignment is to introduce you to **Tailwind CSS**, a modern utility-first CSS framework that allows developers to build responsive and beautiful user interfaces directly within their HTML and JSX.

Moving forward, Tailwind CSS will be our primary styling solution because it:

- Speeds up UI development.
- Reduces the need for writing custom CSS.
- Encourages consistency in design.
- Makes responsive design easier.
- Is widely used in modern React and Next.js applications.

By the end of this assignment, you should be comfortable installing, configuring, and using Tailwind CSS in React applications.

## Assignment Instructions

### Step 1: Fork the Repository

Fork this repository to your own GitHub account.

---

### Step 2: Clone Your Fork

Clone your forked repository to your local machine.

```bash
git clone <your-forked-repository-url>
```

---

### Step 3: Install Dependencies

Navigate into the project directory and install all dependencies.

```bash
npm install
```

---

### Step 4: Run the Project

Start the development server.

```bash
npm run dev
```

Confirm that the application runs successfully before proceeding.
